## CANVA STEP-BY-STEP

1. Go to [canva.com](https://canva.com) → **Create a design** → **Video** → **1920 × 1080**
2. Upload all screenshots (S1–S20) at once via **Uploads**
3. For each slide: drag screenshot as background → resize to fill → add text boxes per the caption specs above
4. For label pills: Insert → **Shape → Rounded Rectangle** → set color → place text on top
5. **Transitions**: Click between slides → select "Slide" direction Left (0.3s)
6. **Music**: go to **Audio** tab → search "cyberpunk" → add to timeline
7. **Export**: Share → **Download** → MP4 Video → 1080p

## CAPCUT STEP-BY-STEP

1. New Project → Import all screenshots
2. Drag to timeline in slide order above
3. Select each clip → set duration (tap the number)
4. **Text**: Add Text → type caption → position bottom-center → apply bold white style
5. **Transitions**: tap between clips → pick "Slide" 0.3s
6. **Auto Captions**: if recording voiceover, use Auto Captions feature
7. **Export**: 1080p 60fps
